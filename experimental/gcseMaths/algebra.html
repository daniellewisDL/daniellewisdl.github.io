2.1 Algebraic Notation

💡 Key Concept

In algebra, letters represent unknown values or variables. Standard conventions:

  • ab means a × b
  • 3y means y + y + y (or 3 × y)
  • a² means a × a
  • Coefficients can be fractions, e.g. ½x

Substitution

Replace letters with numbers to evaluate an expression.

✏️ Worked Example

If a = 3 and b = −2, find the value of 2a² − 3b

Step 1: Substitute: 2(3)² − 3(−2)
Step 2: = 2 × 9 − (−6)
Step 3: = 18 + 6 = 24 ✓

2.3 Simplifying Expressions

💡 Key Concept

Like terms have the same letter(s) raised to the same power. Only like terms can be added or subtracted.

5x + 3y − 2x + y = 3x + 4y

✏️ Worked Example

Simplify: 4a²b − 2ab + 3a²b + 5ab

Step 1: Group like terms: (4a²b + 3a²b) + (−2ab + 5ab)
Step 2: = 7a²b + 3ab ✓

2.4 Expanding Brackets & Factorising

💡 Key Concept

Expanding means multiplying out brackets. Factorising is the reverse — putting brackets in.

Single Bracket

3(2x + 5) = 6x + 15

Double Brackets (FOIL)

(x + 3)(x + 5) = x² + 5x + 3x + 15 = x² + 8x + 15

Factorising Quadratics

Find two numbers that multiply to give the constant term and add to give the coefficient of x.

✏️ Worked Example

Factorise: x² + 7x + 12

Step 1: Need two numbers that multiply to 12 and add to 7
Step 2: 3 × 4 = 12 and 3 + 4 = 7 ✓
Step 3: Answer = (x + 3)(x + 4) ✓

Difference of Two Squares

a² − b² = (a + b)(a − b)

Example: x² − 25 = (x + 5)(x − 5)

Factorising ax² + bx + c Higher

✏️ Worked Example

Factorise: 2x² + 7x + 3

Step 1: Multiply a × c = 2 × 3 = 6
Step 2: Find two numbers that multiply to 6 and add to 7: 1 and 6
Step 3: Split middle term: 2x² + x + 6x + 3
Step 4: Factorise in pairs: x(2x + 1) + 3(2x + 1)
Step 5: = (x + 3)(2x + 1) ✓

2.5 Formulae & Rearranging

💡 Key Concept

To rearrange a formula, use inverse operations to isolate the required variable. What you do to one side, you must do to the other.

✏️ Worked Example

Make t the subject of: v = u + at

Step 1: Subtract u from both sides: v − u = at
Step 2: Divide both sides by a: t = (v − u)/a ✓

Proving Identities Higher

Show that one side of an identity can be manipulated to equal the other side.

✏️ Worked Example

Show that (n + 1)² − (n − 1)² ≡ 4n

Step 1: Expand (n+1)² = n² + 2n + 1
Step 2: Expand (n−1)² = n² − 2n + 1
Step 3: Subtract: (n² + 2n + 1) − (n² − 2n + 1) = 4n ✓

2.6 Functions Higher

💡 Key Concept

A function maps an input to a single output. Notation: f(x) = 2x + 3.

Composite function: fg(x) means apply g first, then f.

Inverse function: f⁻¹(x) reverses the function.

✏️ Worked Example

If f(x) = 2x + 3, find f⁻¹(x).

Step 1: Let y = 2x + 3
Step 2: Swap x and y: x = 2y + 3
Step 3: Rearrange: y = (x − 3)/2
Step 4: f⁻¹(x) = (x − 3)/2 ✓

2.7 Coordinates & Straight-Line Graphs

💡 Key Concept

The equation of a straight line is y = mx + c, where m is the gradient (steepness) and c is the y-intercept (where it crosses the y-axis).

gradient m = (y₂ − y₁) / (x₂ − x₁) = rise / run
y = 2x + 1

Parallel & Perpendicular Lines

Parallel lines have the same gradient. If line 1 has gradient m, any parallel line also has gradient m.

Perpendicular lines Higher have gradients that multiply to −1. If one line has gradient m, the perpendicular gradient is −1/m.

Example: Line has gradient 3. Parallel line: gradient = 3. Perpendicular line: gradient = −1/3.

✏️ Worked Example

Find the equation of the line through (1, 5) and (3, 11).

Step 1: Gradient m = (11 − 5)/(3 − 1) = 6/2 = 3
Step 2: Substitute into y = mx + c: 5 = 3(1) + c → c = 2
Step 3: Equation: y = 3x + 2 ✓

2.8 Quadratic Functions

💡 Key Concept

A quadratic function has the form y = ax² + bx + c. Its graph is a parabola (U-shape if a > 0, ∩-shape if a < 0).

Roots are where the parabola crosses the x-axis (y = 0). The turning point is the vertex.

y = x² − 2x − 3

Completing the Square Higher

x² + bx + c = (x + b/2)² − (b/2)² + c

✏️ Worked Example

Write x² + 6x + 2 in completed square form.

Step 1: Half of 6 is 3, so (x + 3)²
Step 2: (x + 3)² = x² + 6x + 9
Step 3: x² + 6x + 2 = (x + 3)² − 9 + 2 = (x + 3)² − 7 ✓
Turning point is at (−3, −7) ✓

2.10 Graphs in Real-Life Context

💡 Key Concept

Graphs can model real situations. A distance-time graph shows how position changes over time. The gradient equals speed. A horizontal line means stationary.

Speed-Time Graphs

Gradient = acceleration. Area under the graph = distance travelled.

Estimating Gradients and Areas Higher

For curved graphs, draw a tangent at the point to estimate the gradient. Use the trapezium rule to estimate the area under a curve.

2.11 Equation of a Circle Higher

💡 Key Concept

The equation of a circle centred at the origin with radius r is:

x² + y² = r²

Example: x² + y² = 25 is a circle with centre (0,0) and radius 5.

The tangent to a circle is perpendicular to the radius at the point of contact.

2.12 Solving Linear & Quadratic Equations

💡 Key Concept

Solving means finding the value(s) of the unknown that make the equation true.

Linear Equations

✏️ Worked Example

Solve: 3(2x − 1) = 4x + 7

Step 1: Expand: 6x − 3 = 4x + 7
Step 2: Subtract 4x: 2x − 3 = 7
Step 3: Add 3: 2x = 10
Step 4: Divide by 2: x = 5 ✓

Solving Quadratics by Factorising

✏️ Worked Example

Solve: x² − 5x + 6 = 0

Step 1: Factorise: (x − 2)(x − 3) = 0
Step 2: Either x − 2 = 0 → x = 2, or x − 3 = 0 → x = 3
Solutions: x = 2 or x = 3 ✓

The Quadratic Formula Higher

x = (−b ± √(b² − 4ac)) / 2a

2.13 Simultaneous Equations

💡 Key Concept

Two equations with two unknowns. Solve by elimination or substitution.

✏️ Worked Example — Elimination

Solve: 2x + 3y = 12 and 4x − 3y = 6

Step 1: Add the equations (3y and −3y cancel): 6x = 18
Step 2: x = 3
Step 3: Substitute into first equation: 2(3) + 3y = 12 → 3y = 6 → y = 2
Solution: x = 3, y = 2 ✓

Linear/Quadratic Simultaneous Equations Higher

Substitute the linear equation into the quadratic to form a single quadratic equation.

2.14 Iteration Higher

💡 Key Concept

Iteration uses a formula repeatedly to approach a solution. Given xₙ₊₁ = f(xₙ) and a starting value x₀, calculate x₁, x₂, x₃, … until values converge.

✏️ Worked Example

Use xₙ₊₁ = (5 + xₙ)^(1/3) with x₀ = 2 to find a solution to x³ − x = 5.

x₁ = (5 + 2)^(1/3) = 7^(1/3) ≈ 1.9129
x₂ = (5 + 1.9129)^(1/3) ≈ 1.9063
x₃ ≈ 1.9058 — converging to x ≈ 1.906

2.15 Inequalities

💡 Key Concept

Solve inequalities just like equations, BUT when you multiply or divide by a negative number, flip the inequality sign!

✏️ Worked Example

Solve: 3x − 7 < 2x + 4

Step 1: Subtract 2x: x − 7 < 4
Step 2: Add 7: x < 11 ✓

Number Line Representation

Open circle ○ for < or > (not included). Filled circle ● for ≤ or ≥ (included). Shade the valid region.

Quadratic Inequalities Higher

Solve by factorising the quadratic, finding the roots, and using a sketch to determine the solution region.

2.16 Sequences

💡 Key Concept

A sequence is an ordered list of numbers following a rule. The nth term formula gives you any term directly.

Types of Sequences

Finding the nth Term of a Linear Sequence

✏️ Worked Example

Find the nth term of: 5, 8, 11, 14, …

Step 1: Common difference = 3, so it starts 3n
Step 2: When n = 1: 3(1) = 3, but first term is 5. Adjust: +2
Step 3: nth term = 3n + 2 ✓

Quadratic Sequences Higher

If second differences are constant (say = 2a), the nth term has the form an² + bn + c.
